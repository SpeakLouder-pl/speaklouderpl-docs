Make.com is useful for two cases:

1. sending leads to campaign,
2. receiving post call data from SpeakLouder.

<img src="/images/app-integracje.png" alt="Integrations page with Make card" />

## Typical setup

- In Make, create webhook scenario.
- In SpeakLouder, add URL where needed.
- Test with one lead before full traffic.

For workflow details, see [/guides/workflow-and-scenario](/guides/workflow-and-scenario).
