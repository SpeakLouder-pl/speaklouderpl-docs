## Where to edit workflow

Open campaign, then **Automatyzacja**, then **Edytuj scenariusz**.

<img src="/images/app-kampania-automatyzacja.png" alt="Automation page in campaign" />

## Scenario editor

The editor has two areas:

- left side, scenario path,
- right side, settings for selected module.

<img src="/images/app-scenariusz-edytor.png" alt="Scenario editor view" />

## Main modules

- **Start**, fixed start point.
- **Połączenie**, call step.
- **SMS**, text message step.
- **Koniec**, fixed end point.

## After call settings

When you select **Połączenie**, you can set:

- phase name,
- SMS after call,
- action after call, for example transfer or summary email.

<img src="/images/app-scenariusz-polaczenie-panel.png" alt="Call module settings in right panel" />

## Good practice

Start simple:

1. one call step,
2. one delay,
3. optional SMS,
4. save and test.
