## Menu główne (po zalogowaniu)

| Etykieta | Do czego służy |
| -------- | -------------- |
| **Przegląd** | Pulpit startowy, skrót do ostatnich kampanii, podsumowania. |
| **Moje kampanie** | Lista kampanii, tworzenie nowej, wejście do wybranej. |
| **Integracje** | Połączenie z Meta Lead Ads, Gmail, Make.com z poziomu konta. |
| **Kredyty** | Saldo na rozmowy i SMS, historia doładowań i zużycia. |
| **Ustawienia** | Ustawienia konta (nie mylić z ustawieniami pojedynczej kampanii). |

Możesz też mieć widok **Leady** poza pojedynczą kampanią — część pracy i tak robi się **wewnątrz wybranej kampanii**.

## Menu wewnątrz kampanii

| Etykieta | Do czego służy |
| -------- | -------------- |
| **Pulpit** | Skrót informacji tylko o tej kampanii. |
| **Formularze** | Kampanie z reklam (np. Meta): wybór formularza leadowego. |
| **Źródło** | Kampanie z pliku / importu albo technicznego kanału: mapowanie kolumn itd. |
| **Agent** | Instrukcje i głos dla rozmów przychodzących i wychodzących; test z przeglądarki. |
| **Numer telefonu** | Identyfikator numeru z systemu telefonii (odbiór i wychodzące). |
| **Leady** | Lista osób w tej kampanii, wyszukiwanie, karta leada. |
| **Połączenia** | Widok skupiony na rozmowach w tej kampanii. |
| **Automatyzacja** | Scenariusz: oczekiwania, połączenia, SMS, opcjonalnie webhook po rozmowie; historia i harmonogram. |
| **Pomoc** | Krótkie samouczki po polsku w kontekście tej kampanii. |
| **Ustawienia** | Ustawienia samej kampanii (np. godziny dzwonienia, wstrzymanie). |

**Uwaga:** tytuł karty przeglądarki czasem może różnić się od etykiety w menu (stare ścieżki). W razie wątpliwości **ufaj menu po lewej**.
