## Menu główne

| Etykieta | Do czego służy |
| --- | --- |
| Przegląd | Główny podgląd i podsumowania |
| Moje kampanie | Lista kampanii i tworzenie kampanii |
| Integracje | Podłączenie narzędzi zewnętrznych |
| Kredyty | Saldo, zużycie, historia transakcji |
| Ustawienia | Ustawienia konta |
| Pomoc | Otwiera docs.speaklouder.pl w nowej karcie |

## Menu kampanii

| Etykieta | Do czego służy |
| --- | --- |
| Pulpit | Podsumowanie kampanii |
| Formularze lub Źródło | Ustawienie źródła leadów |
| Agent | Głos i zachowanie rozmowy |
| Numer telefonu | Numer używany przez kampanię |
| Leady | Lista leadów i karta leada |
| Połączenia | Historia rozmów |
| Automatyzacja | Edytor scenariusza i czasy |
| Ustawienia | Ustawienia tej kampanii |
| Pomoc | Otwiera docs.speaklouder.pl w nowej karcie |
