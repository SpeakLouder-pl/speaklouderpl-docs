The product uses **Polish labels** in the interface. Below, the Polish name is first and the English meaning is in parentheses.

## Main menu (always visible after login)

| Polish label | English meaning | Purpose |
| ------------ | ---------------- | ------- |
| **Przegląd** | Overview / home | Welcome dashboard, quick access to recent campaigns, summary charts. |
| **Moje kampanie** | My campaigns | Create, rename, and open campaigns. |
| **Integracje** | Integrations | Connect Meta Lead Ads, Gmail, or Make.com to your account. |
| **Kredyty** | Credits | Balance for calls and SMS, history of usage and top-ups. |
| **Ustawienia** | Settings | Account-level preferences. |

There is also a **Leads** entry in some layouts for a cross-campaign list; day-to-day work is often **inside one campaign**.

## Inside a campaign (left menu)

| Polish label | English meaning | Purpose |
| ------------ | ---------------- | ------- |
| **Pulpit** | Dashboard | Short summary for this campaign only. |
| **Formularze** | Forms | For **Meta-style** campaigns: pick which lead ad form feeds this campaign. |
| **Źródło** | Source | For **file / feed-style** campaigns: import or map columns, or manage inbound technical source. |
| **Agent** | Agent | Instructions and voice for **incoming** and **outgoing** calls; browser test call. |
| **Numer telefonu** | Phone number | ID of the number from your telephony setup used for in and out calls. |
| **Leady** | Leads | Searchable list of people; open one for stage and timeline. |
| **Połączenia** | Calls | Call-focused view for this campaign. |
| **Automatyzacja** | Automation | Visual scenario: waits, calls, SMS, optional post-call webhook; history and scheduled actions. |
| **Pomoc** | Help | In-app tutorials (Polish). |
| **Ustawienia** | Settings | Campaign-specific options (for example allowed calling hours, pause). |

Note: Browser tab titles may occasionally differ from menu labels for legacy routes; trust the **sidebar** names when they disagree.
