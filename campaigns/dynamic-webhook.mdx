description: Campaign setup for technical webhook integrations.

This type is for teams that send leads programmatically.

<Warning>
  This flow usually needs technical support, because it uses webhook URL, payload fields, and token security.
</Warning>

## Business view

- Your team still works in the same places, **Leady**, **Automatyzacja**, **Połączenia**.
- Difference is only how leads enter campaign, through webhook instead of form or file.

## If you are non technical

Ask your implementation partner to configure webhook, then use the campaign normally.
