Use this type if you want to call or message people after abandoned cart events.

## Setup path

1. Create campaign for Shopify flow.
2. Connect Shopify source according to your account setup.
3. Set workflow timing and call strategy.
4. Monitor results in **Leady** and **Połączenia**.
